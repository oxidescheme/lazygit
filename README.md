## Installation

Lazygit config files live here:

- Linux: `~/.config/lazygit/config.yml`
- macOS: `~/Library/Application Support/lazygit/config.yml`
- Windows: `%APPDATA%\lazygit\config.yml`

Copy the contents of [`themes/oxide.yml`](./themes/oxide.yml) into your `config.yml`, then restart lazygit.
